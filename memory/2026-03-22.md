### Key Lessons
- Gaussian blur on dim mask → center value only 0.3, dimming too weak. Use hard/feathered mask.
- Binary hard mask looks like black circles. Use soft mask with tiny blur (3px) instead.
- Glow adds brightness even at dim states — don't add glow below certain brightness threshold.
- `brightness**2` dimming curve too aggressive. Linear dimming at 50% min looks natural.
- Always render AND deploy after code changes — editing the script doesn't update Vercel!
- Per-light radius scaled by y-position gives realistic perspective depth effect.
